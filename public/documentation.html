myFlix movie_api Documentation

The endpoints sorted after the used HTTP method. POST, GET, PUT DELETE (CRUD).

Business Logic URL HTTP Method Request body data format Response body data format
Add a user /users POST A JSON object holding data about a user to add, structured like:
{
"name": "Arno"
}		 A JSON object holding data about user that was added, including an ID:
{
id: 1,
name: "Arno",
favoriteMovies: []
}
Add a movie to user's favorite movies array /users/:id/:movieTitle POST none A message indicating whether the movie was successfully added
Get a list of all movies /movies GET none A JSON object holding data about all the movies
Get data about a single movie by title /movies/:title GET none A JSON object holding data about the movie
Get data about a genre /movies/genre/:genreName GET none A JSON object holding data about the genre
Get data about a director /movies/directors GET none A JSON object holding data about the director
Find user by ID and update user name /users/:id PUT A JSON object holding data about a user to change name, structured like:
{
"name": "Arno"
}		 A JSON object holding data about a user's changed name:
{
id: 1,
name: "Arno",
favoriteMovies: []
}
Remove a user by ID /users/:id DELETE none A message indicating whether the user was successfully removed
Remove a movie from user's favorite movies array /users/:id/:movieTitle DELETE none A message indicating whether the movie was successfully removed
